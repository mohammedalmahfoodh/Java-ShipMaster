To run KSMBRTU special features. (from version 1.1.625)
Put a file in the app-path folder named KSMSPini.txt

Example of file contents (2 lines):

"AlarmValue","13412"
"AlarmRetry","3"


Details:

AlarmValue - Alarm limit to compare in raw data for one DAU only with FC=16
AlarmRetry - Number of retries before alarm is displayed (all individual alarms are logged)
